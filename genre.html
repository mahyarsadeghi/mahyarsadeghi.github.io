Genre & Artist Analysis



Top 15 genres VS number of artist + others

This first horizontal bar chart shows the top 15 genres considering the number of artists that belong to each of these genres.
By hovering on each bar, it is possible to see the exact amount of artis for the desired genre.




TOP 15 genres VS sales amount and TCU units + others

The second horizontal bar chart shows the top 15 genres considering
the total amount of sales ($) for each of these genres, or the total amount of TCU (unit), based on the clicked button.
By hovering on each bar, it is possible to see the exact sales' value ($) or the TCU's units for the desired genre.




TOP 20 artists for sales amount and TCU units

Here we can see the top 20 best selling music artists of all time,
differentiating them by sales amount ($) and TCU (unit), based on the performed selection.
By hovering on each bar, it is possible to see the exact sales' value ($) or the TCU's units for the specific artist.




TOP best selling music artist for each country

The last chart of this section is a bubble chart. Each bubble represents an artist, specifically the best selling music artist for each of the country
In the sales configuration there are the best selling artists considering the sale amount (y-axis), but the size of the bubbles are proportional to the TCU of each artist.
In the TCU scenario there are the best selling artists considering the TCU (y-axis). If this is option is selected, the size of the bubbles will be proportional to the sales amount of each artist.
By hovering on each bar, it is possible to retrieve some meaningful info about an artist.
